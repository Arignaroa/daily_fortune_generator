# 생각토론 3파트 오늘의 운세 (GitHub Pages)

정적 HTML 한 파일로 동작합니다.

## 배포 방법 (GitHub Pages)
1) 이 폴더의 `index.html`을 GitHub 저장소(repo) 루트에 업로드  
2) GitHub 저장소에서 **Settings → Pages**  
3) **Deploy from a branch** 선택  
4) Branch: `main` / Folder: `/ (root)` → Save  
5) 배포 URL로 접속: `https://<github-id>.github.io/<repo-name>/`

## 로컬 실행
- 그냥 더블클릭으로 열어도 됩니다.
- 모바일 테스트는 크롬에서 `chrome://inspect` 또는 개발자도구 디바이스 모드 추천.

## 참고
- Tailwind는 CDN을 사용하므로 인터넷 연결이 필요합니다.
- ‘공유하기’는 모바일/브라우저 지원 여부에 따라 동작이 달라질 수 있습니다.
